Welcome to FLEX's documentation!
===================================

**FLEX**  is a Python library for Federated Learning
that helps the user to simulate a federated learning
environment. FLEX is designed to federate models, and
to adapt easily a centralized model to a federated 
environment.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
